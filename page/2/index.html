Ubuntu常用 apt-get

Posted on 2016-09-28   |   In operating system

The apt-get utility is a powerful and free package management command line program

upgrade 是升级已安装的所有软件包，升级之后的版本就是本地索引里的，因此，在执行 upgrade 之前一定要执行 update, 这样才能是最新的。

update 是同步 /etc/apt/sources.list 和 /etc/apt/sources.list.d 中列出的源的索引，这样才能获取到最新的软件包。

apt-get update updates the list of available packages and their versions, but it does not install or upgrade any packages.
apt-get upgrade actually installs newer versions of the packages you have.

apt-get update 更新软件列表。
apt-get upgrade 更新软件。
apt-get dist-upgrade 更新系统版本。

Backpack series and Dynamics Programming

Posted on 2016-09-25   |   In alog

Get down stairs

There are 10 stairs, every times you can only walk 2 or 3 stairs. How many ways to exactly reach the 10th stairs.

10 floor, you can step up 2 or 3, how many ways to arrive the 10th from 0th

Backpack Issue 1:

n items, every items volume is (Integer) A[i] (1 <= i <=n) sorted?, every backpack’s volumn is integer m. What is the maximum volume of items can the backpack hold?

sample inputs: n = 4 A=[2,3,5,7] m=11

brutal-force

n steps, every step have two options . 50 items 2^50

Thinking:

whether to put the last item (to put 7)?

if put the question become that to obtain the max volume of remaining for a backpack of 11- 7 = 4

else, it become that to obain max volume of remaining for a backpack of 11

So in math, we represent like f(4，11) = max(f(3，11),f(3，4)+7)

f(i，j) = max(f(i-1，j-A[i])+A[i]， f(i-1，j))

f(0) Puzzle

For all DP problem, do not start from zero. put f(0,j) as an Initialization

the first parameter always be i - 1, because no matter u put or not always need to exclude the last element

f(i) only has the relationship of f(i - 1), so

f(1) f(0)

f(2) f(1)

…….

The range, 1 <= i <= n; 0 <= j <= m

The Time Complexity O(M*N): n = A.length ; m= size of the backpack

Initialization:

f(0,0) = 0; f(0,1) = 0;……… f(0,m) = 0;

0 item put in 0 volume backpack; 0 item put in 1 volume backpack; 0 item put in 2 volume backpack ……

Backpack Issue 2 add Value

n items, every items volume is (Integer) A[i] (1 <= i <=n) sorted?, every volume is (integer) V[i] (0<=i<=n). The backpack’s volumn is a constant integer m. What is the maximum volume of items can the backpack hold?

n element, the volume of every element is A[i] (1 <= i <=n ),

Value is V[i] (1 <= i <n). The volume of one backpack is m. How much value will the backpack contains？

Sample input: n = 4，A=[2,3,5,7]，V=[1,5,2,4]，m=11

Thinking:

whether to put the last item (to put 7)?

if put the question become that to obtain the max volume of remaining for a backpack of 11- 7 = 4

else, it become that to obain max volume of remaining for a backpack of 11

So in math, we represent like f(4，11) = max(f(3，11)，f(3，11-4)+4)

f(i，j) = max（f(i-1，j - A[i]) + V[i]， f(i - 1，j)）
